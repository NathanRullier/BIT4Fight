But du programme : créer un fighting game fonctionnant avec 4 boutons en commençant par 2

Contrôles : 
le principe est que le joueur utilise une flèche directionnelle pour le mouvement. Quand il appuie simultanément sur l'autre flèche directionnelle (dans le sens inverse ) il se met à frapper
Le joueur peu enchaîner des coups ( Combo ) en relâchant et réappuyant assez vite sur la flèche opposée à la direction.

(Pressed)LeftArrowKey : Run
	RightArrowKey : Punch1
	RightArrowKey : Punch2
	RightArrowKey : Punch3
	RightArrowKey : Punch4

(Pressed)RightArrowKey : Run
	LeftArrowKey : Punch1
	LeftArrowKey  : Punch2
	LeftArrowKey  : Punch3
	LeftArrowKey  : Punch4
